How to Use It

  1. Swap [icon] with your desired icon name - like Shield, User, Key, Email, or Firewall.
  2. Paste the JSON into your image generation tool.
  3. Adjust primary/accent color if needed (for example, neon green for access, red for alert).
  4. Generate - and get instantly usable, transparent-background icons.
